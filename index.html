Wikontor on GitHub

My User Page

Willkommen auf der GitHub-Seite von Wikontor.

Diese Seite wurde automatisch erzeugt mit dem Github Page-Generator. Das ist ein einfacher Weg, um auf Github schöne Seiten für die eigenen Projekte zu erstellen. Die Seiteninhalte können hier gepflegt werden. Einfach ein vorbereitetes Template auswählen und loslegen. Nachdem eine Seite erstellt ist, kannst du den neuen gh-pages Branch lokal auschecken. Wenn du GitHub Desktop verwendest, kannst du dein Repository einfach synchronisieren und siehst dann deinen neuen Branch.

Designer Templates

Github stellt einige schöne Templates zur Verfügung. Du kannst vor der Veröffentlichung der Seite eines von ihnen ansehen und auswählen. Auch noch nach der Veröffentlichung der Seite kannst du den Page-Generator erneut anwenden und zu einem anderen Theme wechseln. Dein Seiteninhalt bleibt dabei unverändert.

Seiten manuell erstellen

Wenn du Seiten nicht über den Page-Generator erstellen willst, lege einen Branch mit der Bezeichnung gh-pages in deinem Repository an, um eine Seite manuell zu erstellen. Zusätzlich zu regulärem HTML unterstützt GitHub Pages auch Jekyll, einen einfachen , für Blogs geeigneten statischen Site-Generator. Jekyll macht es einfach, seitenübergreifende Header und Footer zu erstellen, ohne dass diese manuell in die Seiten kopiert werden müssen. Auch einige weitere intelligente Blog- und Template Funktionen werden unterstützt.

Autoren und Unterstützer

Du kannst einen GitHub User-Namen @mention, um einen Link zu diesem Profile zu erstellen. Der resultierende <a> Verweis zeigt auf das GitHub-Profil des Unterstützers. Als Beispiel: Im Jahr 2007 haben Chris Wanstrath (@defunkt), PJ Hyett (@pjhyett) und Tom Preston-Werner (@mojombo) GitHub gegründet.
